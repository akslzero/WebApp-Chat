# Realtime Chat WebApp

Website aplikasi chat sederhana.

## Fitur

- ✅ Authentication (Login/Register/Logout)
- ✅ Realtime chat
- ✅ Add Friends
- ✅ Responsive (mobile & desktop)

Coming Soon

- Delete Message
- Upload file/image
- Call

## Tech Stack

### Frontend

- React 18 + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui components
- Socket.io Client
- React Router
- Axios

### Backend

- Node.js + Express
- DB MySQL
- JWT Authentication
- Socket.io
